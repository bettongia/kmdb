# v0.09

## GitHub Actions signing pipeline

- [ ] **macOS**: Developer ID certificate + `notarytool` notarization for
      `betto_zstd.dylib` and `kmdb_cli`. Store certificate and App Store Connect
      API key as Actions secrets. ORT dylib (from Microsoft) is pre-signed.
- [ ] **Windows**: Authenticode OV certificate. ⚠ EV certificates are
      hardware-bound and impractical in CI — investigate cloud HSM signing as an
      alternative before treating Windows as unblocked.
- **iOS / Android**: Signing handled by Xcode / Flutter toolchain; no additional
  pipeline work for native libraries.
